# Developer Tools Project

This project explores browser developer tools and their various features for web development debugging and analysis.

## Tasks Overview

- Responsive device testing
- CSS manipulation
- Element state forcing
- Style inspection and copying
- DOM manipulation
- Performance auditing
- Accessibility testing
- Security analysis

## Website Used
https://dev-tools.alx-tools.com/

## Tools Used
- Chrome DevTools / Firefox Developer Tools
- Device toolbar for responsive testing
- Elements panel for DOM inspection
- Console for JavaScript execution
- Network panel for request analysis
- Lighthouse for auditing
